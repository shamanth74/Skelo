# Skelo Lubricants Website

A modern, responsive business website built for **Skelo Lubricants**, showcasing premium motorcycle engine oils and lubrication solutions.

This project focuses on delivering a clean brand experience, high performance, and search engine optimization for better online visibility.

---

## Live Website

https://skelolubricants.in

---

## Project Overview

The Skelo Lubricants website is designed to highlight the brand's premium lubricant products, communicate its mission, and provide customers with a smooth and professional browsing experience.

The site emphasizes:

* High-performance product presentation
* Modern UI/UX design
* Fast loading speed
* SEO optimization
* Mobile responsiveness

---

## Features

* Fully responsive design for mobile, tablet, and desktop
* SEO optimized metadata and structured data
* Smooth scroll animations and UI transitions
* Modern design using Tailwind CSS
* Fast static hosting via GitHub Pages
* Custom domain integration
* Secure HTTPS deployment

---

## Tech Stack

* HTML5
* Tailwind CSS
* JavaScript
* GitHub Pages (Hosting)
* Custom Domain Configuration

---

## Deployment

The website is deployed using **GitHub Pages** and connected to the custom domain:

https://skelolubricants.in

Any changes pushed to the `main` branch automatically redeploy the website.

---

## SEO Optimization

The website includes:

* Optimized meta tags
* Open Graph tags for social media sharing
* Twitter card support
* Structured data (Schema.org)
* Robots.txt and sitemap readiness
